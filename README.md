# Twilio TUI

A terminal UI for managing Twilio services -- SMS, calls, phone numbers, logs, and account profiles.

Built with TypeScript, [Bun](https://bun.sh), and [OpenTUI](https://opentui.com).

## Features

- **SMS** -- view messages and compose new ones
- **Calls** -- call history and outbound dialing
- **Numbers** -- manage, search, and purchase phone numbers
- **Logs** -- live debugger log viewer with severity coloring
- **Accounts** -- switch profiles and view usage

## Prerequisites

- [Bun](https://bun.sh) installed
- [Twilio CLI](https://www.twilio.com/docs/twilio-cli/quickstart) installed and configured (`twilio login`)

## Getting Started

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```
